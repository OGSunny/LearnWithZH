Frame & Button Forge

StarterGui > ScreenGui > Frame > TextButton. UDim2 for flex, UICorner for crisp edges. TextScaled keeps labels crisp on all devices.

Core Button Build (LocalScript)

local Players = game:GetService("Players")
local player = Players.LocalPlayer
local playerGui = player:WaitForChild("PlayerGui")

local screenGui = Instance.new("ScreenGui")
screenGui.Name = "MyCoolGui"
screenGui.Parent = playerGui

local frame = Instance.new("Frame")
frame.Size = UDim2.new(0, 250, 0, 120)
frame.Position = UDim2.new(0.5, -125, 0.5, -60)
frame.BackgroundColor3 = Color3.fromRGB(45, 55, 72)
frame.BorderSizePixel = 0
frame.Parent = screenGui

local corner = Instance.new("UICorner")
corner.CornerRadius = UDim.new(0, 12)
corner.Parent = frame

local button = Instance.new("TextButton")
button.Size = UDim2.new(1, 0, 1, 0)
button.Text = "Launch Adventure"
button.TextColor3 = Color3.new(1,1,1)
button.TextScaled = true
button.BackgroundTransparency = 1
button.Parent = frame

button.MouseButton1Click:Connect(function()
    print("Adventure awaits!")
    -- Fire remote or tween something fun
end)

UIListLayout Setup

local listLayout = Instance.new("UIListLayout")
listLayout.FillDirection = Enum.FillDirection.Vertical
listLayout.SortOrder = Enum.SortOrder.LayoutOrder
listLayout.Padding = UDim.new(0, 5)
listLayout.Parent = frame

-- For grid
local gridLayout = Instance.new("UIGridLayout")
gridLayout.CellSize = UDim2.new(0, 100, 0, 100)
gridLayout.Parent = frame

Make 'Em Dance

Tweens for that buttery feel. Enter: scale up, color shift. Leave: fade out gentle. 2025: Use EasingStyle.Sine for natural bounces.

Tween Magic

local TweenService = game:GetService("TweenService")

local enterInfo = TweenInfo.new(0.3, Enum.EasingStyle.Back, Enum.EasingDirection.Out)
local enterProps = {
    BackgroundColor3 = Color3.fromRGB(100, 181, 246),
    Size = UDim2.new(1.05, 0, 1.05, 0)
}

local leaveInfo = TweenInfo.new(0.2, Enum.EasingStyle.Quad, Enum.EasingDirection.In)
local leaveProps = {
    BackgroundColor3 = Color3.fromRGB(52, 152, 219),
    Size = UDim2.new(1, 0, 1, 0)
}

button.MouseEnter:Connect(function()
    TweenService:Create(button, enterInfo, enterProps):Play()
end)

button.MouseLeave:Connect(function()
    TweenService:Create(button, leaveInfo, leaveProps):Play()
end)

Perf Pro Tips

Single ScreenGui per purpose. Reuse instances for backgrounds/buttons. ScreenInset: Legacy for buttons—avoids core UI overlap. 2025: Use UIStroke for lightweight borders.

  • Pool tweens—don't create per hover.
  • Aspect constraints for cross-device glory.
  • Instance pooling for dynamic lists.
