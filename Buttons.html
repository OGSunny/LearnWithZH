    Frame & Button Forge

    StarterGui > ScreenGui > Frame > TextButton. UDim2 for flex, UICorner for crisp edges. TextScaled keeps labels crisp on all devices.

    Launch Adventure

    Basic Button Setup

    local button = script.Parent -- TextButton
button.Size = UDim2.new(0, 200, 0, 50)
button.Position = UDim2.new(0.5, -100, 0.5, -25)
button.Text = "Launch Adventure"
button.TextScaled = true
button.BackgroundColor3 = Color3.fromRGB(45, 55, 72)
button.BorderSizePixel = 0

local corner = Instance.new("UICorner")
corner.CornerRadius = UDim.new(0, 8)
corner.Parent = button

    Tween Tactics: Smooth Hovers

    TweenService for buttery animations. EaseInOutQuad for natural feel. 2025 tip: Animate BackgroundTransparency for subtle feedback.

    Hover Tween

    local TweenService = game:GetService("TweenService")

local hoverTween = TweenService:Create(button, TweenInfo.new(0.2, Enum.EasingStyle.Quad, Enum.EasingDirection.InOut), {Size = UDim2.new(0, 220, 0, 55), BackgroundColor3 = Color3.fromRGB(79, 70, 229)})

button.MouseEnter:Connect(function()
    hoverTween:Play()
end)

button.MouseLeave:Connect(function()
    local leaveTween = TweenService:Create(button, TweenInfo.new(0.2, Enum.EasingStyle.Quad, Enum.EasingDirection.InOut), {Size = UDim2.new(0, 200, 0, 50), BackgroundColor3 = Color3.fromRGB(45, 55, 72)})
    leaveTween:Play()
end)

    Perf Pro Tips

    Single ScreenGui per purpose. Reuse instances for backgrounds/buttons. ScreenInset: Legacy for buttons—avoids core UI overlap. 2025: Use UIStroke for lightweight borders.

    • Pool tweens—don't create per hover.
    • Aspect constraints for cross-device glory.
    • Instance pooling for dynamic lists.
